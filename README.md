# FlowCanvas v2
Stack: React + Vite + Bootstrap 5 + Supabase

## Cambios v2
- Login/Registro con nombre de usuario (username)
- Dashboard individual por cliente
- Bootstrap 5 integrado como base
- Diseño responsive mejorado

## Setup
1. Ejecuta schema.sql en Supabase SQL Editor
2. Crea .env con VITE_SUPABASE_URL y VITE_SUPABASE_ANON_KEY
3. npm install && npm run dev
